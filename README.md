🎓 Stream Recommendation System for 10th Pass-Out Students

📌 Project Overview

Choosing the right academic stream after 10th grade is an important decision for students. This project builds a machine learning-based Stream Recommendation System that helps 10th pass-out students select the most suitable stream based on their academic performance.

The model analyzes students' marks and recommends an appropriate stream such as Science, Commerce, or Arts.

🎯 Objective

The main objective of this project is to assist students in making better academic decisions by providing data-driven stream recommendations using machine learning.

📊 Dataset

The dataset was created using students’ 10th class academic scores, collected from schools.
The dataset contains subject-wise marks that are used as input features for training the model.

Example features may include:

Mathematics Marks

Science Marks

English Marks

Social Science Marks

Overall Percentage

Target variable:

Recommended Stream (Science / Commerce / Arts)

🤖 Machine Learning Model

Random Forest Classifier

The Random Forest algorithm is used to build the recommendation model.

Reasons for choosing Random Forest:

Handles classification problems effectively

Reduces overfitting

Works well with structured data

Provides high accuracy
